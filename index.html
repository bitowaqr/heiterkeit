// Research

Hei | ter | keit

noun, f. — Cheerfulness, mirth, amusement. German parliamentary stenographers note it in parentheses — (Heiterkeit) — when the chamber is amused: a quiet, dignified form of collective laughter, distinct from (Lachen), which is louder and often directed at someone.
An analysis of one million recorded reactions during Plenary sessions of the German Bundestag 1998-2026.
1,027,516 annotations 1,699 sessions 1998–2026 WP 14–21
// Volume

Parliament gets louder

Recorded reactions per session more than doubled since 1998 — from ~400 to over 1,000.

// Mood

Less laughter, more shouting

Amusement (Heiterkeit + Lachen) divided by confrontation (Zurufe + Widerspruch) - something happens around 2013...

// Heckling

The heckling

The Grand Coalition in 2013 triples heckling. AfD in 2017 doubles it again.

// Heiterkeit

Signal vs. Noise

Heiterkeit per session barely changed over 25 years. But the explosion of Zurufe (=Heckling) buried it.

// Häme

Not all laughter is funny

Lachen surged after 2017. The AfD accounts for 40–60% of all recorded laughter since WP19. Often directed at other MPs.

// Ratio

The quality of laughter

Heiterkeit as share of all laughter: peaked at 75% in the mid-2010s, below 40% by 2026. The Bundestag still laughs — but differently.

// Method

How this was built

1,027,516 annotations from 1,690 plenary sessions (WP14–21, 1998–2026), extracted programmatically from official Bundestag Plenarprotokolle. WP18–21 from structured XML via dserver.bundestag.de, WP14–17 from the DIP API full-text endpoint.

Built with Claude Code (Anthropic) for data pipeline, statistical analysis, and visualisation.
